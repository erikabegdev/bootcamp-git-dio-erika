# 🚀 Bootcamp GitHub DIO - Erika  

Repositório criado para registrar os **desafios e práticas** do Bootcamp de Git/GitHub da [Digital Innovation One](https://www.dio.me/).  
Inclui comandos básicos, exemplos e anotações de estudo para **versionamento de código**.  

---

## 📚 Conteúdo  

- ✅ Clonando repositórios (`git clone`)  
- ✅ Adicionando arquivos (`git add`)  
- ✅ Criando commits (`git commit -m`)  
- ✅ Enviando alterações para o GitHub (`git push`)  
- ✅ Outros comandos úteis do dia a dia  

---

## 🛠 Tecnologias Utilizadas  

- Git  
- GitHub  
- Markdown  

---

## ✨ Objetivo  

Praticar versionamento de código, documentação com Markdown e consolidar os conceitos de Git e GitHub para uso em projetos reais.
